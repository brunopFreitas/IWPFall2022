CONTRAST

Contrast is important in web design because:

It is important to make different things look different.

Important elements must stand out over less important elements.

It creates pages where the eyes flow naturally.

REPETITION

Repetition is important in web design because:

It is important to create a sense of consistency on the page.

It provides visual cues to the user on how to follow the content on the page.

It creates also a sense of cohesiveness in a design.

ALIGNMENT

Alignment is important in web design because:

It creates a sense of organization on the page.

It helps to connect other elements on the page.

It allows you to create the correct content flow of the page.

PROXIMITY

Proximity is important in web design because:

It creates a connection between elements on the page.

It helps to keep the content coherent on the page.

It helps the user to understand what is being presented on the page.
